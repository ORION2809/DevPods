DevPods Relay UI Prototype

Built from docs/17-end-user-developer-mobile-ui-spec.md. Visual direction: Material 3 structure with transparent liquid-glass components, calm technical audio-control feel, user-first Standard Mode with Dev Mode progressively disclosed. This version adds explicit UX state coverage for settings, offline queue, autonomy, approval details, setup lifecycle, bridge management, permissions, diagnostics, notification controls, QR failure, compatibility, accessibility, localization, empty states, and success confirmations.

Home = product state Activity = transcript + approvals Device = pairing + capability truth Help = recovery + diagnostics Dev = operator console State screens = trust and recovery
